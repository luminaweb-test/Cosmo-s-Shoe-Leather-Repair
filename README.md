# Cosmo's Shoe & Leather Repair

Premium single-page static website for Cosmo's Shoe & Leather Repair in North Syracuse, NY.

## Files

- `index.html` is the full website.
- `assets/leather-stitch-texture.png` is a local texture image used in the hero and map panel.

## GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open Settings > Pages.
3. Set the source to the `main` branch and `/root`.
4. Save, then open the GitHub Pages URL once it finishes deploying.

No build step is required. The page uses Tailwind through the CDN so it can be hosted directly from GitHub Pages.
